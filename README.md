# Course for graph data structure and searching algorithm


### Require software
1. CMake
2. Ninja
3. gcc

### Configure the Ninja build file
```bash
./configure.sn
```

### Build the ARM platform executed file
```bash
./build.sn
```

### Run the cmake ctest
```bash
./run_test.sh
```
